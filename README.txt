HOW TO USE:
  Open index.html in any browser (double-click or drag to Chrome/Firefox).
  No server needed for most features (Google Maps embed may need HTTPS).

FIGHTER TEAM ACCESS:
  Hover over the animated logo in the navigation.
  A "◈ RESTRICTED ACCESS" hint will appear.
  Click the logo → red terminal hacking sequence activates → fighter-team.html loads.
  Press ESC at any time to cancel the sequence.
